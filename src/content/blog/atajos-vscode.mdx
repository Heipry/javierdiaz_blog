### **Acelera tu Desarrollo: Los 25 Atajos de VS Code Que Todo Programador Debería Conocer**

Como desarrolladores, pasamos horas frente al editor de código, y cada segundo cuenta. Si bien las extensiones son fantásticas, a menudo subestimamos el poder de los atajos de teclado. Conocer los atajos correctos puede transformar tu productividad, permitiéndote navegar y editar tu código a la velocidad del pensamiento.

Aquí he recopilado una lista de los 25 atajos esenciales de **VS Code**, muchos de los cuales he utilizado en mi día a día y que han demostrado ser verdaderos cambios de juego. No son trucos de magia, sino herramientas que, una vez dominadas, se sienten como una extensión natural de tus manos.

#### **Edición y Manipulación de Código**

Empecemos por lo más básico: mover, copiar y editar líneas de código de forma instantánea, sin necesidad de cortar y pegar.

* **Mover líneas**: ¿Necesitas reordenar una función o un bloque de código? Con `Alt + ↑` o `↓` puedes mover la línea o selección actual sin esfuerzo.
* **Copiar líneas**: Similar al anterior, pero para duplicar. Usa `Shift + Alt + ↑` o `↓` para crear una copia idéntica de una línea o bloque de código.
* **Seleccionar apariciones**: Es uno de mis favoritos. Con `Ctrl + D`, seleccionas la siguiente aparición de la palabra o variable que tienes resaltada. ¿Quieres cambiar el nombre de una variable en todo el archivo? Usa `Ctrl + Shift + L` para seleccionar **todas las apariciones** de una vez.

Además de las selecciones simples, puedes editar en múltiples puntos a la vez.

* **Múltiples cursores**: Para añadir varios cursores de forma vertical, usa `Ctrl + Alt + ↑` o `↓`. Si necesitas más flexibilidad, `Alt + Click` te permite añadir un cursor en cualquier punto de tu archivo.

Para bloques de código más grandes, la manipulación también es fluida.

* **Mover/Copiar bloques**: Selecciona el bloque que quieres mover y utiliza `Alt + ↑` o `↓` para reubicarlo. Si prefieres duplicarlo, el atajo es `Shift + Alt + ↑` o `↓`.
* **Borrar línea**: Olvídate de seleccionar y borrar. Simplemente posiciona el cursor en la línea que quieres eliminar y presiona `Ctrl + Shift + K`.

#### **Navegación y Búsqueda Inteligente**

Navegar por un proyecto grande puede ser un caos. Estos atajos te ayudarán a moverte con agilidad entre archivos y definiciones.

* **Ir a definición**: ¿Quieres saber de dónde viene una función o variable? Posiciónate sobre ella y presiona `F12`.
* **Ojear definición**: Si solo quieres echar un vistazo sin cambiar de archivo, usa `Shift + F12`. Verás una pequeña ventana emergente con la definición del código.
* **Abrir/Crear archivo**: Con `Ctrl + Click` sobre un nombre de archivo en tu código, puedes abrirlo directamente. Si el archivo no existe, VS Code te preguntará si quieres crearlo.


#### **Visibilidad y Entorno de Trabajo**

A veces, la clave para concentrarse es eliminar las distracciones.

* **Ocultar barra lateral**: Usa `Ctrl + B` para ocultar o mostrar la barra lateral de archivos.
* **Zen Mode**: Para una inmersión total en tu código, activa el `Zen Mode` con `Ctrl + K Z`. Elimina todas las distracciones de la interfaz, dejando solo tu código en el centro.

#### **Gestión de la Terminal**

La terminal es una herramienta indispensable. Usa estos atajos para gestionarla sin sacar las manos del teclado.

* **Ocultar/Mostrar terminal**: `Ctrl + '` (o `Ctrl + ñ` en teclados españoles) es tu atajo para el control de la terminal.
* **Nuevo terminal**: Si necesitas una nueva instancia, `Ctrl + Shift + '` (o `Ctrl + Shift + ñ`) es tu amigo.

#### **Atajos de Composición y Organización**

Estos atajos te ayudarán a estructurar tu código y a trabajar con diferentes archivos al mismo tiempo.

* **Comentar línea**: El clásico `Ctrl + \` (o `Ctrl + Shift + 7` en teclados españoles) para comentar o descomentar una línea.
* **Comentar selección**: Para comentar un bloque de código, selecciónalo y presiona `Shift + Alt + A`.
* **Cambiar de tab**: Navegar entre pestañas abiertas es rápido con `Ctrl + TAB`. Puedes ir directamente a la pestaña anterior con `Ctrl + PG UP` o a la siguiente con `Ctrl + PG DN`.
* **Cerrar tab/todas**: `Ctrl + W` cierra la pestaña actual. Para cerrar todas las que tienes abiertas, el atajo es `Ctrl + K Ctrl + W`.
* **Reabrir tab anterior**: ¿Cerraste algo por error? `Ctrl + Shift + T` lo reabrirá al instante.

#### **Funcionalidades Avanzadas**

Finalmente, algunos atajos para tareas más específicas que te harán sentir como un verdadero profesional.

* **Ir a línea**: ¿Necesitas ir a la línea 150? Presiona `Ctrl + G`, escribe el número de línea y presiona `Enter`.
* **Ver definiciones ordenadas**: Con `Ctrl + P`, escribe `@:` y verás una lista de todas las definiciones (funciones, clases, etc.) de tu archivo actual.
* **Atajo Emmet**: Para expandir abreviaciones de Emmet, usa `Ctrl + Shift + P` y busca `Emmet: Expand Abbreviation`.
* **Cambiar ocurrencias de un símbolo**: Selecciona un símbolo (variable, función, etc.) y presiona `F2`. Podrás cambiar su nombre en todas las apariciones de tu proyecto.
* **Ir a la configuración**: Para acceder a la configuración de VS Code de manera rápida, usa `Ctrl + ,`.

#### **Un Apunte Importante**

Si alguno de estos atajos no te funciona, no te preocupes. Puede ser que tu configuración de teclado sea diferente o que una extensión lo esté sobrescribiendo. Puedes verificar y editar cualquier atajo con `Ctrl + K S`. Esta es una herramienta poderosa que te permite personalizar VS Code a tu gusto, ajustando los atajos para que se adapten perfectamente a tu flujo de trabajo.

Si quieres tener todos estos atajos a mano de forma rápida, hemos preparado una "chuleta" que puedes guardar. La tienes disponible en este gist: [Gist con todos los atajos de VS Code](https://gist.github.com/Heipry/0c585a989953f46d56792cb2d93c0a15).
